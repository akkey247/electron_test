# 基本形

## 構成

```
.
├─ node_modules/
├─ src/
│  ├─ index.html
│  └─ script.js
├─ .gitignore
├─ main.js
├─ gulpfile.js
├─ webpack.config.js
├─ package.json
└─ README.md
```

## 使い方

以下のコマンドで実行。

```
$ npm run exec
```

以下のコマンドでビルド。

```
$ gulp build
```

以下のコマンドでパッケージ化。

```
$ gulp package:win32
```
